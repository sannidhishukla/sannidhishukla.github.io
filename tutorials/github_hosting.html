Starting Out
  1. Head over to GitHub and create your account. You'll have to enter some basic information like your username and email address.
    Tip: for your username, use your full name or something distinctive and easy to remember
  2. Verify your email address and you'll be good to go!
  3. Download the correct version of GitHub desktop for your operating system. Once the download is complete, log in using the username and password you just created.
  4. If you haven't already, set up your project files within one folder to avoid broken pathways. This way, when you set up your repository folder you'll be able to move them over without any problem.
Creating a Repository
  1. Open up GitHub desktop and sign in. In the upper left-hand corner of your screen click File and then New Repository.
    Tip: Mac users can also user the keyboard shortcut Cmnd + N to open up the option to create a new repository.
  2. Use [username].github.io, the name of the project folder, as the name for the repository.
  3. When asked for the local path, choose where on your computer you would like your repository to be created. GitHub will create a new empty folder for your site here with the name [username].github.io.
  4. Click create repository to create your first GitHub repository!
Using your Repository and Publishing
  1. At this point you have two important folders â€” your working folder, where you've set up your project files so far, and your repository folder that GitHub has just set up. Move or copy all of the project files from your working folder to your repository.
  2. Open GitHub desktop back up. In the left-hand sidebar you should see all of your project files listed as changes. Below the list of files, in the lower left-hand corner, type in a brief summary and description of the changes you've made to your repository.
    Tip: It may not seem important now to type in a meaningful description of your changes but as you move forward and continue making changes and additions to your site, it'll be helpful to know what you did when.
  3. Below, click Commit to master. Under the button you should get a small message that tells you that your changes have been committed.
  4. Click 'Publish repository' at the top of the page. Your site should be up and running now!
  5. Visit [username].github.io, share it with your friends and family, put it on business cards, whatever! The internet is your oyster now, captain.
